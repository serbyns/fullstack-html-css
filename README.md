# HTML + CSS Course Projects

Цей репозиторій містить практичні завдання з курсу HTML + CSS GoIT.
Кожен модуль — це окремий етап розвитку одного сайту.

## Структура

- `module-01-html-basics` — основи HTML, семантична розмітка
- `module-02-css-introduction` — базові стилі, normalize, шрифти, кольори
- `module-03-layout-flexbox` — Flexbox, блокова модель, відступи, контейнери
- `module-04-decorative-effects` — SVG-іконки, hover, тіні, оверлеї
- `module-05-forms-accessibility` — форми, модальне вікно, доступність (a11y)
- `module-06-responsive-design` — адаптивна верстка, Mobile First, breakpoints, мобільне меню

## Технології

- HTML5
- CSS3
- Flexbox
- Mobile First
- SVG Sprite
- modern-normalize
